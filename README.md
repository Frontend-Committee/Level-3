#  X Clone

A **clone of X** built with **HTML, CSS, and JavaScript**, featuring interactive social media functionalities with **persistent data using localStorage**.

---

##  Pages

| Page | Description |
|------|-------------|
| `index.html` | Main feed showing all posts |
| `explore.html` | Explore new content |
| `notifications.html` | View your notifications |
| `bookmarks.html` | See all bookmarked posts |

---

##  Features

-  **Add Post** – Create new posts dynamically  
-  **Like &  Repost** – Interact with posts  
-  **Comment** – Add comments on any post  
-  **Bookmark** – Save posts to view later  
-  **Theme Switcher** – Toggle between light and dark mode  
-  **Persistent Data** – All posts and interactions are saved in `localStorage`  

---

##  Technologies Used

- **HTML5** – Page structure  
- **CSS3** – Styling, responsive design, and themes  
- **JavaScript** – DOM manipulation, event handling, localStorage  


---

## How It Works

1. **Posts**: Stored in localStorage and displayed instantly.  
2. **Like / Repost / Comment**: Updates both UI and localStorage.  
3. **Bookmarks**: Clicking bookmark saves the post and displays it on the bookmarks page.  
4. **Theme Switcher**: Toggle dark/light mode across all pages.
